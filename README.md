# Ayush Subedi — Portfolio

Personal portfolio website built with React + Vite.

**Live site:** https://subayush.vercel.app

## Tech Stack
- React 18 + Vite
- React Router v6
- EmailJS (contact form)
- CSS Modules
- Deployed on Vercel

## Features
- Responsive design (mobile + desktop)
- Working contact form
- Project showcase
- Skills overview

## Local Development
npm install
npm run dev
